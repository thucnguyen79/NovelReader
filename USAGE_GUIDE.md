# 📘 Hướng Dẫn Sử Dụng NovelReader

## 1. Thêm Sách Mới

1. Mở app → Nhấn nút **"+ Thêm Sách"** trên trang chủ
2. Nhập **tên sách** (bắt buộc)
3. Dán nội dung truyện vào ô **"Nội dung"**
4. Nếu truyện có nhiều chương, dùng **ký tự phân cách** (ví dụ: `---` hoặc `Chapter`) để app tự tách chương
5. Nhấn **"Lưu"**

## 2. Cấu Hình Dịch Thuật (Gemini API)

1. Vào tab **Cài đặt** (⚙️)
2. Nhập **API Key** từ [Google AI Studio](https://aistudio.google.com/)
3. Nhấn **"Kiểm tra API Key & Tìm Model"**
   - App sẽ liệt kê tất cả model mà API Key hỗ trợ
   - Chọn model phù hợp (khuyến nghị model có chữ "flash")
4. Nhấn **"Lưu Cài Đặt"**

> 💡 **Mẹo**: Nếu gặp lỗi "limit: 0", hãy thử model khác. Mỗi API Key có quyền truy cập khác nhau tùy theo khu vực.

## 3. Dịch Truyện

### Dịch từng chương
- Vào **Chi Tiết Sách** → Nhấn icon 🌐 bên phải chương muốn dịch

### Dịch tất cả
- Vào **Chi Tiết Sách** → Nhấn nút **"Dịch tất cả"** ở trên cùng

### Dịch lại
- **Từ Chi Tiết Sách**: Nhấn icon 🔄 bên cạnh ✅ của chương đã dịch
- **Từ trang Đọc**: Nhấn icon 🔄 cạnh toggle Bản dịch/Bản gốc

## 4. Đọc Truyện

1. Chạm vào chương bất kỳ để mở trang đọc
2. Sử dụng toggle **"🇻🇳 Bản dịch"** / **"📄 Bản gốc"** để chuyển đổi
3. Chạm vào vùng text để ẩn/hiện thanh điều khiển
4. Dùng nút **−/+** trên header để tăng/giảm cỡ chữ

### Chỉnh sửa nội dung
1. Nhấn icon ✏️ trên header
2. Nội dung chuyển thành ô soạn thảo
3. Chỉnh sửa xong → Nhấn **"Lưu"** hoặc **"Hủy"**

> ⚠️ Bạn có thể edit cả bản gốc lẫn bản dịch. Chọn đúng tab trước khi nhấn ✏️.

## 5. Text-to-Speech (Đọc To)

### Điều khiển cơ bản
| Nút | Chức năng |
|-----|-----------|
| ⏹ | Dừng hoàn toàn |
| ⏪ | Lùi 1 câu |
| ▶️/⏸ | Phát / Tạm dừng |
| ⏩ | Tiến 1 câu |

### Tùy chỉnh nâng cao
Nhấn nút **▲** để mở bảng tùy chỉnh:
- **Tốc độ**: Kéo thanh trượt (0.5x → 2.0x)
- **Giọng đọc**: Chọn giọng Tiếng Việt hoặc Tiếng Anh
- **Hẹn giờ tắt**: 15 / 30 / 45 / 60 / 90 phút

> 💡 Câu đang được đọc sẽ được **highlight** trên màn hình.

## 6. Sắp Xếp Chương

1. Vào **Chi Tiết Sách**
2. Dùng nút **▲/▼** bên trái mỗi chương để di chuyển lên/xuống
3. Thứ tự mới được lưu tự động

## 7. Giao Diện

Vào **Cài đặt** → **Giao Diện** → Chạm để chuyển đổi:
- **Hệ thống**: Theo cài đặt điện thoại
- **Sáng**: Luôn nền trắng
- **Tối**: Luôn nền tối
